LOVE AND GREENS

PRESENTAION

I have created a responsive food blog. I used "Wordpress" to provide content from an REST API. For this porject we were only allowed to use HTML, CSS and JavaScript.

This is a design for a healthy food blog. This site gives you healthy recepies and tips on how to start your healthy life journey. I chose colorful pictures and the colour mint green for this project because the colour green in food resebles healthyness and freshness.

PROCESS

I always start my projects by creating some kind of sketches, either on paper or in my mind. Through out this whole processn i learned tremendous of knowledge. I really enjoyed working with JavaScript beacuase JS is a whole world and you can do so much with the help of JS.

Even though this project was really exciting and fun to do, i faced some complications, for example with the carousel slider on the front page. It was difficult to make beacause i had never created one carousel slide before. But problem solving is the key! I Love to solve problems i face because i have a mind set that whenever i face a problem i think that "now i will learn something new". I felt so proud over my self when i finished this project. This is my best project so far.

EVALUATION

When i look back at this project i feel satisfied with this project, but if i reconsider i could make a few changes. First of all, all the white-space around the carousel slider seems a little bit too and that could have been less. The subscribe email section might be placed more to the right. otherwise i am really satisfied with this site.
